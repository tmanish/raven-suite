Welcome to Raven Write

This is your canvas — a full-featured word processor with the polish of a native Mac app. Everything you'd reach for is in the floating toolbar above: bold, italic, underline, colors, highlights, lists, tables, images, and links.

A few things to try

  • Select text and hit ⌘B, ⌘I, or ⌘U — or press ⌘F to find & replace.
  • Use the style menu to switch between Title, Heading, Quote, and Code block.
  • Hover the table button for a drag-to-size grid picker.
  • Flip on dark mode with the sun icon up top. The paper follows.
  • Export to Word, HTML, Markdown, or PDF from the Export menu.
The quill is dead. Long live the raven.

Select all of this and delete it when you're ready to write something real. Your work autosaves as you type.
